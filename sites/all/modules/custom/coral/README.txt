Coral Module
======================
by Adrian Webb, adrian.webb@coraltech.net



Description
===========

Provides a framework and API for building and managing server stacks and 
clusters from Drupal.

The Coral module provides a framework and API for creating, testing, and
managing Puppet modules and server stacks.  

This includes:

It shares a core configuration system designed to build extensible tree based 
parameters for use as Puppet parameters and JSON configurations.



Installation
============

1. Download, extract and copy the coral module to your sites/all/modules directory.
2. Enable the "Coral" module on your Drupal Modules page, under the Administration heading.
